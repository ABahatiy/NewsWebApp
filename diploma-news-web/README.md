# Diploma News (Next.js + Vercel)

Це стартовий каркас вебзастосунку-стрічки новин (як “агрегатор”), який:
- має UI на Next.js (App Router) + Tailwind
- має API: `GET /api/news` (підтягує RSS і повертає нормалізований JSON)
- підтримує фільтр тем і пошук
